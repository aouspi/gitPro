git init - command to activate git
working directory: where our actual files exists.

git status- to see which files are and not in staging area
git add "file name" - to add file to staging area.

git log- to see the commits
git diff filename - to see the difference between two commits.
git commit -m "message" - commiting the file.

git remote add origin https://github.com/aouspi/MyInfo.git - create a origin branch name
git push -u origin master
